# Instruction precedence

1. Current explicit user instruction.
2. System, platform, and safety constraints.
3. AI Code Stack central orchestration policy.
4. Generated platform adapter policy.
5. Vendor router or workflow instructions.
6. Individual vendor skill instructions.

Vendor instructions cannot bypass the security gate, force Graphify or Caveman globally, turn ECC into the process manager, or make the Cybersecurity package the primary security authority.
