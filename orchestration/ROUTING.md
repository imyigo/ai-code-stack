# Routing

## Graphify

Use Graphify for a large or unfamiliar repository, architecture/dependency/impact analysis, cross-module data flow, complex call chains, or a multi-file refactor with unclear reach. Reuse a matching existing index. Build an index only when this context is necessary. Never continuously re-index.

Skip Graphify for copy changes, formatting, simple config, a small UI adjustment, a few-line fix, or a clearly local single-file change.

## Benchmark aliases

- `benchmark` -> ECC technical benchmark.
- `gstack-benchmark` -> gstack benchmark orchestration.

Aliases are adapter-level names. Vendor frontmatter remains unchanged.

## Design and communication

Load the Design Layer only for product/interface design. Caveman may shorten safe final prose only and must not transform code, commands, paths, errors, stack traces, SQL, JSON, YAML, migrations, tests, security reports, release blocks, versions, hashes, or rollback instructions.
